# JavaScript的设计模式


<hr/>

设计模式并不是结构、框架，不能生搬硬套。光记住其概念也没用，我们平时写的代码中可能就在不经意间有一些设计模式的影子。而你没有发觉！

这里以最简单清晰地代码总结一些设计模式的demo。对着它发呆，总会悟出点什么。
争取用最简单的代价、花最少的时候能够对设计模式有一些感悟。

设计模式的万能定律就是解决代码中变化的部分与不变的部分的结构和关系。
